KING OS V5007 FRONTEND CLEAN UPLOAD

用途：
- 修正前台五語頁滿版顯示。
- 保留首頁 index.html。
- 保留後台 admin.html。
- 套件內不含任何 -2.html 檔案。

最穩上傳方式：
1. 解壓縮本 ZIP。
2. 打開解壓後資料夾。
3. 全選裡面的所有檔案與資料夾。
4. 到 GitHub Kingos-5001 → Add files → Upload files。
5. 拖入／選取全部檔案。
6. Commit changes。

注意：
- 不要上傳 ZIP 本身。
- 不要在 Working Copy 使用「匯入檔案」，它容易產生 -2.html。
- 如果 GitHub 上已經有 -2.html，請手動刪除那些 -2.html，或不要理會；網站不會讀取它們。
